# React Analytics Dashboard

Dashboard administrativo moderno desarrollado con React, Tailwind CSS y Recharts para visualización de métricas, estadísticas y análisis empresariales mediante gráficos interactivos y componentes reutilizables.

---

## Descripción

Proyecto frontend desarrollado como panel administrativo tipo analytics dashboard, enfocado en visualización de datos, diseño responsive y experiencia moderna de usuario.

El sistema implementa métricas dinámicas, gráficos interactivos, tablas de ventas, filtros por período y soporte para modo oscuro.

## Ejecutar proyecto

Instalar dependencias:

```bash
npm install
```

Ejecutar entorno de desarrollo:

```bash
npm run dev
```

Abrir en navegador:

```text
http://localhost:5173
```
