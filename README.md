# React Analytics Dashboard

Dashboard administrativo desarrollado con React, Tailwind CSS y Recharts para visualización de métricas, estadísticas y gráficos interactivos.

---

## Tecnologías utilizadas

- React
- Vite
- Tailwind CSS
- Recharts
- JavaScript

---

## Funcionalidades planeadas

- Dashboard de métricas
- Tarjetas estadísticas
- Gráficos interactivos
- Filtros dinámicos
- Diseño responsive
- Componentes reutilizables
- Sidebar de navegación

---

## Ejecutar proyecto

Instalar dependencias:

```bash
npm install
```

Ejecutar entorno de desarrollo:

```bash
npm run dev
```

Abrir en navegador:

```text
http://localhost:5173
```
